
# GitHub User Search App

## Overview

The GitHub User Search App allows users to search for GitHub users and view their profiles along with a list of their repositories. This app leverages the GitHub API to fetch and display user data.

## Features

- **User Search:** Enter a GitHub username to search for a user.
- **User Profile Display:** Shows the user's avatar and login name.
- **Repositories List:** Displays a list of public repositories of the user with links to view them on GitHub.

## Technologies Used

- **HTML5** for structuring the web page.
- **CSS3** for styling and animations.
- **JavaScript** for API interaction and dynamic content updates.
